On an $N$ by $N$ grid, Arav wants a laser going horizontally through $(1, 1)$ to pass through $(N, N)$ for his science project. To that end, he has placed some mirrors on the grid to redirect the laser to its destination.

He has just realized, however, that his configuration of mirrors doesn't actually reflect the laser to $(N, N)$! Even worse, he glued all the mirrors to the grid and now they can't be moved.

Find the minimum number of new mirrors Arav must add to finish his science project.

Input

The first line contains $N$ $(1 \le N \le 10^3)$ - the size of the grid.
The next $N$ lines contain $N$ characters each, with '_' for an empty tile and '/' , '\' for a mirror.
It is guaranteed that there is a valid solution.

Output

Output $N$ lines, each with $N$ characters. Each line should be the configuration of the grid after the mirrors have been placed. The solution should contain the minimum number of mirrors. If there are multiple solutions with the minimum number of mirrors, output any of them.

Sample Input
5 ____\ _\___ ____/ _____ _\___ Sample Output
____\ _\___ _/__/ _____ _\___

Note

Note that $(1, 1)$ is located at the top left corner and $(N, N)$ is at the bottom right corner.

Subtask 1: $N \le 20$
Subtask 2: No additional constraints
