You have a circle divided into $N$ equally-sized sectors (like cutting a pizza). Each sector has a counter $c_i$, which initially starts at $0$.

In a single operation, you can draw a line through the circle, and each sector the line passes through the interior of gets their counter incremented by $1$.

What is the minimum number of operations to make the counters of all sectors equal to exactly $K$?

Input

The first line of input contains a single integer $T$ ($1 \le T \le 10^6$) - the number of independent test cases.

The next $T$ lines of input each contain $N$ and $K$ ($1 \le N, K \le 10^9$) - the number of sectors and the desired counter respectively.

Output

For each test case, print a single integer on a new line - the minimum number of operations to make the counters of all sectors equal to $K$.

Sample Input
3 1 3 3 2 4 1 Sample Output
3 2 2

Note

For the third sub-test case, we can draw one line going through sectors $1$ and $3$, and one line going through sectors $2$ and $4$.

Subtask 1: No additional constraints
