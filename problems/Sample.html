You are given an array of $N$ integers ($1\le N\le10$ and $0\le a_i\le100$) starting at $i=1$ and ending at $i=N$. You must find the result of running the operation $$\sum_{i=1}^{N}a_i$$ on the array.

Input
The first line contains $N$.
The second line contains the $N$ space-separated integers of the array.

Output
Print one number, the result of the operation on the array.

Sample Input
6 5 82 38 51 24 11
Sample Output
211
