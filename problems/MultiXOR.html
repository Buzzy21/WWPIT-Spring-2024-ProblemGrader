Suvanth is learning about the XOR operation (denoted by $\oplus$) in his AP CSA class. To test him, his teacher told him to replace each number in the array $a$ of length $N$ by the xor of every other number.

Formally, in the new array $v$, $v_i = a_1\oplus a_2\oplus ...\oplus a_{i-1}\oplus a_{i+1} \oplus ... \oplus a_N$. Now replace $a$ with $v$.

Suvanth found this problem to be trivial, so you have to do this operation $k$ times on the array $a$ and print the result.

Input

The first line contains 2 integers, $N$ and $k$ $(1 \le N \le 10^5, 1 \le k \le 10^9)$.
The next line contains $N$ integers $a_1,a_2,...,a_N$ $(1 \le a_i \le 10^9)$.

Output

The first line should contain $N$ integers, the array $a$ after performing the operation $k$ times.

Sample Input
6 1 4 5 2 1 9 7 Sample Output
8 9 14 13 5 11

Note

$5\oplus 2\oplus 1\oplus 9\oplus 7 = 8$
$4\oplus 2\oplus 1\oplus 9\oplus 7 = 9$
$4\oplus 5\oplus 1\oplus 9\oplus 7 = 14$
$4\oplus 5\oplus 2\oplus 9\oplus 7 = 13$
$4\oplus 5\oplus 2\oplus 1\oplus 7 = 5$
$4\oplus 5\oplus 2\oplus 1\oplus 9 = 11$

Since $k = 1$, this operation is only done once so that is the answer

Subtask 1: $N \le 10^3, k \le 10$
Subtask 2: $k \le 10$
Subtask 3: No additional constraints
