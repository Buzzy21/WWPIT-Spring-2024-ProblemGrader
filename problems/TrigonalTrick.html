Consider an equilateral triangle of side length $N$, filled with unit equilateral triangles. Each unit equilateral triangle is initially filled with all $0$s.



In a single operation, you can choose any diagonal of length $d$ and set the values of those triangles to be a permutation of the integers from $1$ to $d$ in any arbitrary order, overwriting any previous values.



A diagonal is defined as a straight segment of triangles where the unit size triangles at the ends of the diagonal touch the larger triangle with one of their full sides.



What is the maximum possible sum you can attain after applying $0$ or more operations?

Input

The first line is 1 integer $T$ $(1 \le T \le 1000)$, the number of testcases.

The first line of each testcase contains one odd integer, $N$ $(1 \le N < 10^{12})$.

Output

Output the maximum possible sum$\mod 10^9+7$.
Sample Input 3 1 3 999999

Sample Output 1 9 916249138

Note

If $N=1$, there is only 1 number in the triangle. If $N=3$, you can have the corners be two $3$'s and one $1$ and center a $2$.

Subtask 1: $N \le 12$
Subtask 2: $N \le 10^6$
Subtask 3: No additional constraints
