Vikram is getting his PhD in quantum computing and for his final experiment, he has built an undirected graph of $N$ nodes and $M$ edges connecting them.

There are also $E$ electrons in nodes in the graph. At every point of time, each electron has an equal probability to go to each of its adjacent nodes. An electron must move each turn. It is possible that multiple electrons are in the same node.

After $T$ time, find the expected number of electrons in each node.

Input

The first line contains 4 integers, $N$, $M$, $E$, and $T$ $(1 \le N \le 100,1 \le M \le \frac{N(N-1)}{2},1 \le E \le 10^5,1 \le T \le 10^{12})$.

The next $M$ lines each contain two integers, $a$ and $b$ $(1 \le a,b \le N, a \neq b)$, the edges of the graph. The graph does not have multi-edges or self-loops.

The next line contains $E$ integers $p_1,p_2,\dots p_E$, the position of each electron $(1 \le p_i \le N)$.

Output

Let the probability be a fraction $\frac{p}{q}$. Output an integer $k$ such that $q*k = p \bmod 10^9+7$
Sample Input
4 4 2 2 1 2 2 3 1 3 3 4 1 2 Sample Output
583333338 583333338 500000004 333333336

Note

The expected values are: $\frac{1}{3}$, $\frac{7}{12}$, $\frac{11}{12}$, $\frac{1}{6}$

Subtask 1: $M \le 100$
Subtask 2: $T \le 1000$
Subtask 3: No additional constraints
