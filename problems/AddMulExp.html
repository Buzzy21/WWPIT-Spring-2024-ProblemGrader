Jie was walking around in HSS when he found a suspicious-looking function with $n$ terms.

$f(x, n) = 1x^1+2x^2+3x^3+...+nx^n$

Given $n$, $x$, and $m$, find $f(x, n)\bmod m$.

Input

The first line contains $T$ $(1 \le T \le 10^5)$, the number of testcases.

The first and only line of each testcase contains the three integers $n$, $x$, and $m$ $(1 \le n, x \le 10^{18}, 1 \le m \le 10^9)$.

Output

For each testcase, print the output of the function.

Sample Input
1 5 2 100 Sample Output
58

Note

$1\cdot 2^1+2\cdot 2^2+3\cdot 2^3+4\cdot 2^4+5\cdot 2^5=258$
$258 = 58 \pmod{100}$

Subtask 1: Sum of $n$ is at most $10^5$, $T \le 200$
Subtask 2: $m$ is guaranteed to be prime, $T \le 200$
Subtask 3: $T \le 200$
Subtask 4: No additional constraints
