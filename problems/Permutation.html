You are given an integer $N$. Call a 0-indexed array $p$ good if $p$ is a permutation of $[0\dots N-1]$ and $i\oplus p_{p_i}=j\oplus p_{p_j}$ for all $(0 \le i < j < N)$.

Count the number of good arrays$\mod 10^9+7$.

Input

The first line contains one integer $T$ $(1 \le T \le 10)$, the number of testcases The next $T$ lines each contain an integer $N$ $(1 \le N \le 10^{6})$.

Output

For each testcase, print the number of good arrays$\mod 10^9+7$.

Sample Input
2 3 4 Sample Output
4 16

The solutions for $N=3$ are $[2, 1, 0]$, $[1, 0, 2]$, $[0, 2, 1]$, $[0, 1, 2]$.

The solutions for $N=4$ are $[3, 2, 1, 0]$, $[2, 3, 1, 0]$, $[2, 1, 0, 3]$, $[3, 1, 2, 0]$, $[1, 2, 3, 0]$, $[1, 3, 0, 2]$, $[1, 0, 3, 2]$, $[1, 0, 2, 3]$, $[3, 2, 0, 1]$, $[2, 3, 0, 1]$, $[2, 0, 3, 1]$, $[0, 3, 2, 1]$, $[0, 2, 1, 3]$, $[3, 0, 1, 2]$, $[0, 1, 3, 2]$, $[0, 1, 2, 3]$

Subtask 1: $N \le 10$
Subtask 2: $N \le 20$
Subtask 3: No additional constraints
