There is an array $a$ of length $N$. Find the maximum value of $gcf(a_i,a_j)\cdot lcm(a_i,a_j)$ for all $i$, $j$ such that $1 \le i < j \le N$.

Input

The first line contains $T$, the number of independent test cases.

For each test case, the first line contains an integer $N$ $(2 \le N \le 10^5)$.

The next line contains the $N$ integers of $a$ $(1 \le a_i \le 10^9)$.

It is guaranteed that the sum of $N$ over all testcases is at most $10^6$.

Output

Output $T$ lines, each containing the answer for that testcase.

Sample Input
1 10 8 1 2 7 12 9 13 14 19 17 Sample Output
323

Note

It is optimal to choose $i = 9$ and $j = 10$.

$$a_i = 19, a_j = 17$$
$$gcf(a_i,a_j) = 1, lcm(a_i,a_j) = 323$$
$$gcf(a_i,a_j)\cdot lcm(a_i,a_j) = 323$$
This is the highest possible answer over all possible $i$ and $j$.

Subtask 1: $a_i \le 10^4$
Subtask 2: sum of $N$ is at most $10^3$
Subtask 3: No additional constraints
